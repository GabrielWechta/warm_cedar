INPUT
Store X
INPUT
Store Y

loop,    Load X
         Subt Y
         Skipcond 800 /if(X>Y) skip
         Jump remainder
         
         Store X
         
         Load quotient
         Add ONE
         Store quotient

         Jump loop 

X, DEC 0
Y, DEC 0
quotient, DEC 0
ONE, DEC 1

remainder,	Skipcond 000 /if(X-Y == 0) jump to full
        jump full

        Load quotient
        Output
        Load X
		Output

		Halt
        
full,   Load quotient
		Add ONE
		Output

		Halt
